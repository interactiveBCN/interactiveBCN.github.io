Qui som?

Interactive BCN som un grup de joves de Barcelona que oferim aplicacions i pàgines web a mida per petites i grans empreses. El nostre àmbit d’especialització són les aplicacions per a mòbils i tauletes així com les pàgines web. També dissenyem software a mida per Windows, Mac i Linux.
