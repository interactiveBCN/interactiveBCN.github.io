¿Quiénes somos?

Interactive BCN somos un grupo de jóvenes catalanes que ofrecemos aplicaciones y páginas web hechas a medida para pequeñas y grandes empresas. Nuestro ámbito de especialización son las aplicaciones para móviles y tabletas así como las páginas web. También diseñamos software a medida para Windows, Mac y Linux.
